## 中文

一个基于 Next.js 的全球城市对比工具，帮助你从收入、生活成本、房价、空气质量、气候等多维度比较世界主要城市，为移居、求职、留学等决策提供数据参考。

### 功能亮点

- 🏙️ **134 座城市** — 覆盖六大洲主要城市
- 💼 **26 种职业** — 按具体职业查看薪资差异
- 💰 **税后收入** — 支持毛收入 / 税后净收入 / 外籍人士净收入三种模式
- 💱 **10 种货币** — 自由切换显示币种（汇率每日自动更新）
- 🌐 **4 种语言** — 中文 / English / 日本語 / Español
- 📊 **多城对比** — 最多 3 城并排对比所有维度
- 🏠 **房价与租金** — 每平米均价、购房年限测算
- 🌿 **空气质量** — AQI (US EPA 标准)
- 🌡️ **气候信息** — 气候类型、月度温度/降水图表、日照时数
- 🔒 **安全/医疗/自由指数** — 多子指标加权综合评分
- 🏆 **城市排行榜** — 按收入、性价比、生活压力等维度排名
- � **自选模式** — 多选指标自定义综合排名，支持气候条件筛选
- �🌙 **深色模式** — 自动跟随系统 / 手动切换

### 快速开始

```bash
git clone https://github.com/qing4132/whichcity.git
cd whichcity
npm install
npm run dev
```

访问 http://localhost:3000

### 技术栈

Next.js 15 · React 18 · TypeScript · Tailwind CSS · Recharts

## English

A Next.js-based global city comparison tool. Compare cities across income, living costs, housing, air quality, climate, and more — useful for relocation, job hunting, and study abroad decisions.

### Features

- 🏙️ **134 cities** across 6 continents
- 💼 **26 professions** with city-specific salaries
- 💰 **Net income** — gross / net / expat net income modes
- 💱 **10 currencies** with daily auto-updated exchange rates
- 🌐 **4 languages** — 中文 / English / 日本語 / Español
- 📊 **Multi-city compare** — up to 3 cities side-by-side
- 🏠 **Housing prices & rent** — per sqm, years-to-buy estimates
- 🌿 **Air quality** — AQI (US EPA standard)
- 🌡️ **Climate data** — type, monthly temp/rain charts, sunshine hours
- 🔒 **Safety / Healthcare / Freedom indices** — weighted composite scores
- 🏆 **City rankings** — by income, value, life pressure, and more
- � **Custom mode** — multi-select metrics for composite ranking with climate filtering
- �🌙 **Dark mode** — auto / light / dark
