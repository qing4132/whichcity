## 中文

一个基于 Next.js 的全球城市对比工具，帮助你从收入、生活成本、房价、空气质量等多维度比较世界主要城市，为移居、求职、留学等决策提供数据参考。

### 功能亮点

- 🏙️ **120 座城市** — 覆盖六大洲主要城市
- 💼 **10 种职业** — 按具体职业查看薪资差异
- 💱 **10 种货币** — 自由切换显示币种
- 🌐 **4 种语言** — 中文 / English / 日本語 / Español
- 📊 **3 种对比模式** — 绝对值 / 相对比例 / 巨无霸指数
- 🏠 **房价数据** — 每平米均价 (USD/m²)
- 🌿 **空气质量** — AQI (US EPA 标准)
- 🌡️ **气候信息** — 气候类型、温度、降水、日照
- 💡 **关键洞察** — 综合推荐、购房年限、决策矩阵
- 🌙 **深色模式** — 支持日间/夜间切换

### 快速开始

```bash
git clone https://github.com/qing4132/citycompare.git
cd citycompare
npm install
npm run dev
```

访问 http://localhost:3000

### 技术栈

Next.js 15 · React 18 · TypeScript · Tailwind CSS · Recharts

### ⚠️ 免责声明

本项目**由 AI 辅助生成**，城市数据（收入、生活成本、房价、空气质量等）均为估算值，**不保证数据的准确性和时效性**。数据仅供参考，不构成任何投资、移居或职业决策建议。请以官方统计数据为准。

## English

A Next.js-based global city comparison tool that helps you compare cities across income, living costs, housing prices, air quality, and more — useful for relocation, job hunting, and study abroad decisions.

### Features

- 🏙️ **120 cities** across 6 continents
- 💼 **10 professions** with city-specific salaries
- 💱 **10 currencies** for flexible display
- 🌐 **4 languages** — 中文 / English / 日本語 / Español
- 📊 **3 comparison modes** — Absolute / Relative / Big Mac Index
- 🏠 **Housing prices** — per sqm in USD
- 🌿 **Air quality** — AQI (US EPA standard)
- 🌡️ **Climate data** — type, temperature, rainfall, sunshine
- 💡 **Key Insights** — overall recommendation, home purchase years, decision matrix
- 🌙 **Dark mode** support

### Quick Start

```bash
git clone https://github.com/qing4132/citycompare.git
cd citycompare
npm install
npm run dev
```

Visit http://localhost:3000
