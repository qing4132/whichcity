二、生活成本重建方案 (零风险)

核心问题：不存在一个全球城市级的免费生活成本数据源。必须分区域拼接政府开放数据。

2.1 各区域数据来源

🇺🇸 美国 (13城) — BLS + Census ACS Public Domain
月生活成本: BLS Consumer Expenditure Survey → 23 个 MSA 有单人年度支出数据 → ÷12 得月成本
URL: https://www.bls.gov/cex/tables/geographic/mean.htm
许可: "everything that we publish... is in the public domain"
纽约示例: 单人月支出 ~$5,500 (2024 数据)
月租: Census ACS Table B25064 (Median Gross Rent) — 城市级
URL: https://data.census.gov/table/ACSDT1Y2023.B25064
许可: Public Domain
纽约示例: Median Gross Rent ~$1,700 (但这是全市中位数包含外围区，市中心 1BR ≈ $3,200-4,500)
房价: Census ACS Table B25077 (Median Home Value) + FHFA HPI
FHFA: https://www.fhfa.gov/data/hpi — Public Domain
可转换为每平米价格: median_value / median_sqft
安全: FBI NIBRS / UCR — 城市级犯罪率
URL: https://cde.ucr.cjis.gov/LATEST/webapp/#/pages/downloads
许可: Public Domain
可计算: 暴力犯罪率 per 100k → 转换为 0-100 安全评分
覆盖: 13/13 美国城市 (100%) — 月成本、月租、房价、安全全部有城市级公开数据
🇪🇺 欧盟+英国+瑞士+挪威 (40城) — Eurostat + 各国统计局 CC BY / OGL
月生活成本: Eurostat HBS (Household Budget Survey) — 国家级家庭支出
+ Eurostat HICP (城市级物价指数) 用于城市内修正
URL: https://ec.europa.eu/eurostat/databrowser/view/hbs_str_t211/default/table
许可: Eurostat 标准许可 (允许商用，需标注来源)
方法: 国家人均支出 × 城市物价指数修正 = 城市估算成本
月租: UK → ONS Private Rental Market Statistics (OGL v3.0 ✅ 允许商用) — 城市级
Germany → Destatis Mietspiegel 参考 (DL-DE/BY-2-0 ✅) — 州级
France → INSEE LOVAC (Licence Ouverte ✅) — 省级
其他 EU → Eurostat HICP-Rent 指数 (相对值, 需要锚点城市)
房价: UK → HM Land Registry PPD (OGL v3.0 ✅) — 城市级实际交易价
欧盟 → Eurostat House Price Index (国家级指数)
各国统计局有更细的数据
安全: UK → Home Office Police Recorded Crime (OGL v3.0 ✅) — 城市级
Eurostat → crim_off_cat 犯罪统计 — 国家级
各国警察统计局通常有城市级数据
覆盖: ~35/40 欧洲城市 (88%) — 西欧/北欧/中欧覆盖好，东南欧部分城市可能只有国家级
🇯🇵🇰🇷🇦🇺🇳🇿🇨🇦 发达国家 (16城) — 各国统计局 政府开放数据
日本 (4城): 总務省統計局 家計調査 — 都道府県級, e-Stat API (CC BY 4.0 日本)
URL: https://www.e-stat.go.jp/
韩国 (2城): KOSTAT 가계동향조사 — 全国+首尔, KOSIS API (公开)
URL: https://kosis.kr/
澳大利亚 (3城): ABS HES — 州级, CC BY 4.0
URL: https://www.abs.gov.au/
新西兰 (2城): Stats NZ HES — CC BY 4.0
URL: https://www.stats.govt.nz/
加拿大 (3城): StatCan SHS — Open Government License (允许商用)
URL: https://open.canada.ca/
覆盖: 14/16 城市有政府开放数据 (台北数据来源需另找)
🌏 发展中国家 (51城) — 缺口最大 覆盖率低
中国 (6城): NBS 统计年鉴 — 有省级消费数据但不清楚是否允许商用再分发。可参考城镇居民人均消费支出
风险: 中国统计数据版权声明模糊
印度 (3城): MOSPI NSSO 消费支出调查 — 公开发布但无明确开放许可
东南亚 (曼谷/胡志明/河内/雅加达/马尼拉/吉隆坡/清迈/巴厘岛/岘港/普吉岛/槟城): 各国统计局数据质量和许可参差不齐
拉美 (墨西哥城/圣保罗/里约/波哥大/利马/圣地亚哥/麦德林/蒙得维的亚等): INEGI(墨西哥)/IBGE(巴西) 有开放数据政策
中东/非洲 (迪拜/多哈/利雅得/内罗毕/开罗等): 政府统计开放度最低
替代方案: World Bank PPP + BigMac 指数估算
对无政府数据的城市，用 World Bank ICP PPP 转换因子 (CC BY 4.0) + BigMac Index (CC BY 4.0) 构建估算:
costEstimate = US_baseline × (PPP_factor × bigMac_ratio × city_weight)
精度: ±25-35%（国家级估算，无法区分同国不同城市）
覆盖: 51/51 (100%)，但精度远低于 Numbeo

三、新安全指数设计 (零风险)

当前方案 (5 子指标)

numbeoSafetyIndex30% → Numbeo 🔴
homicideRateInv25% → UNODC 🔴
gpiScoreInv20% → IEP 🔴
gallupLawOrder15% → Gallup 🟠
wpsIndex10% → Georgetown ✅
4/5 子指标来源不可商用

零风险方案 (4 子指标, 全部 CC BY 4.0 / PD)

WB 凶杀死亡率 (反转)35% → WB CC BY 4.0
WB Political Stability30% → WB CC BY 4.0
WB WGI Rule of Law20% → WB CC BY 4.0
Georgetown WPS Index15% → Georgetown ✅
+ 美国/英国/日本等 14 国: 额外加入城市级犯罪率 (政府 PD 数据)
公式: safetyIndex = 国家级指标×0.6 + 城市犯罪率×0.4
与当前的差异: 丢失了 Numbeo Safety Index（唯一城市级感知指标）和 GPI（综合和平指数）。新方案全部基于客观统计数据（WB/UCDP），无主观感知成分。对于有政府犯罪统计的 14 个国家（~36 城市），可额外加入城市级犯罪率作为调节。
